# Files Directory

## Overview

This directory contains all supporting project documents and non-graphical resources used throughout the development lifecycle of the capstone project.

These files help maintain proper collaboration, documentation, planning, and project organization.

---

## Contents

Examples of files that may be stored here include:

- html
- js ...

---

## Purpose

The purpose of this directory is to centralize important project-related files in one location for easier access and better team collaboration.

---

## Guidelines

- Use clear and descriptive file names
- Avoid duplicate documents
- Update files when changes are made

---

## Important

Do not store graphical assets in this directory.

All images, logos, icons, and design materials should be placed inside the `graphics` directory.
