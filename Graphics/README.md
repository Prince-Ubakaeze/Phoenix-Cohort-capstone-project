# Graphics Directory

## Overview

This directory contains all visual and graphical assets used within the capstone project.

These assets support the application's user interface, branding, and overall visual experience.

---

## Contents

Examples of graphical assets that may be stored here include:

- Logos
- Icons
- UI mockups
- Banners
- Illustrations
- Screenshots
- Design assets
- Background images

---

## Guidelines

- Use optimized image sizes whenever possible
- Use meaningful file names
- Avoid uploading unnecessary duplicate assets
- Prefer SVG format for scalable graphics when applicable
- Keep design versions organized properly

---

## File Naming Convention

Examples:

```bash
main-logo.svg
dashboard-mockup.png
landing-page-banner.jpg
```

---

## Important

Only graphical assets should be stored in this directory.

Project documents and non-visual resources should be placed inside the `files` directory.
